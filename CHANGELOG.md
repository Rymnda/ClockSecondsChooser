# Changelog

## v1 — Initial Release

### Added
- GUI dialog to enable or disable seconds in the Windows system tray clock
- Current status detection (ON / OFF)
- Automatic Dark / Light mode detection
- Keyboard shortcuts:
  - Enter → Enable seconds
  - Esc → Exit
- Automatic Explorer restart (only when required)
- Portable standalone EXE
- Custom application icon

### Fixed
- Proper path handling for EXE context
- Removed startup error dialogs related to null paths

### Notes
This is the first public release.  
Future improvements will be added under the same major version (v1.x).
